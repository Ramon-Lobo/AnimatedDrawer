## How to run this project

### 1. Download or Clone this repository
### 2. Install dependencies
Run on terminal
```
npm install
```
or
```
yarn install
```

#####iOS Specific
Install pods

```
  cd ios && pod install
```

### 3. Run the project
#### Android
```
react-native run-android
```
#### iOS
```
react-native run-ios
```
